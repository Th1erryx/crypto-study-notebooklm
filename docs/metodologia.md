# 🧪 Metodologia

Este projeto foi desenvolvido utilizando o NotebookLM como ferramenta de apoio ao aprendizado.

## Etapas

1. Seleção de fontes sobre criptomoedas
2. Definição de objetivos de estudo
3. Criação de prompts estratégicos
4. Geração de respostas com base nas fontes
5. Organização do conteúdo em documentos
6. Análise crítica do uso da IA

## Abordagem

A IA foi utilizada como ferramenta de aprendizagem ativa, auxiliando na organização e interpretação das informações.

## Conclusão

O uso da inteligência artificial contribuiu para o aprendizado, mas exige pensamento crítico e validação das informações.
