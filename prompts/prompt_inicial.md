# 🎯 Prompt Inicial - Estudo sobre Criptomoedas

## Contexto
Estou estudando investimentos em criptomoedas como pessoa física, com foco em aprendizado inicial e uso responsável de capital.

## Objetivo
Compreender:
- Como iniciar investimentos em criptomoedas
- Principais riscos envolvidos
- Estratégias básicas para iniciantes
- Boas práticas de segurança

## Prompt

Estou pensando na possibilidade de investir em criptomoedas com capital de uma pessoa física iniciante.

Com base nas fontes fornecidas, responda:

1. Qual é a melhor forma de começar?
2. Quais são os principais riscos?
3. Quais estratégias são recomendadas para iniciantes?
4. Como evitar erros comuns?
5. Quais boas práticas de segurança devem ser seguidas?

## Requisitos da resposta

- Utilize apenas as fontes fornecidas
- Estruture a resposta de forma clara e didática
- Separe por tópicos
- Destaque conceitos importantes
- Evite recomendações financeiras absolutas

## Observação

As respostas serão utilizadas para criação de:
- Resumos
- Glossário
- Perguntas estratégicas
- Análise crítica
