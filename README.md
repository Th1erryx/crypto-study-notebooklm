# 📊 Estudo Guiado com IA - Criptomoedas

Projeto desenvolvido utilizando NotebookLM para estudo ativo sobre investimentos em criptomoedas.

## Objetivo
Aplicar IA como ferramenta de aprendizado estruturado.

## Metodologia
- Curadoria de fontes confiáveis
- Geração de perguntas estratégicas
- Respostas baseadas em contexto
- Análise crítica

## Resultados
- Resumos organizados
- Glossário técnico
- Prompts reutilizáveis
