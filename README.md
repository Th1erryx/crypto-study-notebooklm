# 📊 Estudo Guiado com IA - Criptomoedas

## 📌 Descrição

Este projeto foi desenvolvido como um mini guia de estudo utilizando inteligência artificial (NotebookLM) como ferramenta de aprendizagem ativa.

O objetivo foi explorar conceitos introdutórios sobre criptomoedas por meio da curadoria de fontes, engenharia de prompts e organização estruturada do conhecimento.

---

## 🎯 Objetivos

- Compreender o funcionamento básico das criptomoedas
- Identificar riscos associados ao investimento
- Explorar estratégias para iniciantes
- Desenvolver pensamento crítico no uso de IA
- Organizar conhecimento de forma estruturada

---

## 🧠 Metodologia

O projeto foi desenvolvido seguindo as etapas:

1. Curadoria de fontes sobre criptomoedas
2. Definição de objetivos de estudo
3. Criação de prompts estratégicos
4. Geração de respostas com IA (NotebookLM)
5. Estruturação do conteúdo em documentos
6. Análise crítica do uso da IA

A IA foi utilizada como ferramenta de apoio, não como fonte única de verdade.
